C语言可变参数

June 17, 2015

在python中写一个有可变参数的函数或者方法是很容易的，比如下面这个例子:

def print_args(*args, **kwargs):
	print type(args), args
	print type(kwargs), kwargs
	for arg in args:
		print arg
	for arg, value in kwargs.items():
		print '%s = %s' % (arg, value)

>>> print_args(1, 2, 3, x=4, y=5)
<type 'tuple'> (1, 2, 3)
<type 'dict'> {'y': 5, 'x': 4}
1
2
3
y = 5
x = 4
